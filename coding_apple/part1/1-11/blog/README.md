## 🔧 part1-11 props를 응용한 상세페이지 만들기

### 🔹 1. 상세에 데이터를 넘겨줄 때

- 하나의 오브젝트 or 하나의 배열만 보내거나
- index도 같이 보내야함
  => 뭐가 좋을지 상황에 맞춰서 사용하자

- 자식과 부모가 같이쓰는 state라면 부모에 올려놓고 자식이 props로 받아쓰는게 좋다

```jsx
<자식컴포넌트 작명={state}>
function 자식컴포넌트 (props) {
    return (
        {props.state}
    )
}
//부모에서 자식에게만 전송이 가능 (패륜 불륜안됨)
```
